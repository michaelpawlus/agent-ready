# Assets

## demo.gif

The top-of-README animated demo. Record it with [vhs](https://github.com/charmbracelet/vhs):

    vhs assets/demo.tape

That writes `assets/demo.gif`. Commit the result.

`demo.gif` is intentionally not tracked until it's recorded -- GitHub won't show a
broken image, it will fall back to the `alt` text on the `<img>` tag in the README.
